# Monotoniforhold

En kategorisering af hvornår hældningen af en graf en enten stigende, falende aller $0$

| Ligning       | Hældning ($a$)         |
| ------------- | ---------------------- |
| $$f'(x) = 0$$ | Vendepunkter ($a = 0$) |
| $$f'(x) > 0$$ | Positiv ($a > 0$)      |
| $$f'(x) < 0$$ | Negativ ($a < 0$)      |
