# Cirklens Ligning

$$\sqrt{(x-x_0)^2 + (y-y_0)^2} = r$$

**alternativ:**

$$(x-x_0)^2 + (y-y_0)^2 = r^2$$

**Variabler:**
$C = (x_0,y_0)$, cirklens centrum
$P = (x,y)$, punkt på cirklen
$r = |\vec{CP}|$, cirklens radius
