# Libraries in C++


### Locations

- **Headerfiles** should be stored in `/usr/include`
- **Library files** (.so) should be stored in `/usr/lib64` (on Fedora)

### Relevant Flags
- `-I{dir}` add header file directory.
- `-l{library}` Search for a compiled library file (.so)
- `-L{dir}` add .so directory
