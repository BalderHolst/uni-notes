# The C++ Processor

The first part of a c++-file. All commands that interact with the processor are preceded by a '#'.

### Directives

Control flow in the preprocessor.

```c++
#ifndef MY_VARIABLE
#define

#include <import-one>
#include <import-two>
#include <import-three>
#include <import-four>

#endif
```
